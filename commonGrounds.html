Description:

A website built using bootstrap, html, css, firebase, and javascript. I was responsible for developing the logic on this site that calculated the mid-point of the groups and interactions with the api's. The site was designed to allow people to log-in and add friends to a group, as friends were added custom javascript functions would use the data given from group members to calculate the mid-point between the group members and search api's to find restaurants in that location so everyone could travel the same distance to meet up.
